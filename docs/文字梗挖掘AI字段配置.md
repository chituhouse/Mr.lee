# 文字梗挖掘（AI）字段配置指南

> 只为适配度=高或中的话题生成，适配度=低的话题跳过

---

## 配置步骤

### 1. 字段设置
- **字段名称**：文字梗挖掘（AI）
- **字段类型**：AI字段
- **输出类型**：多行文本
- **⚠️ 触发条件**：
  - 勾选「仅当字段满足条件时生成」
  - 条件1：`适配度（AI）` = `高`
  - **或**（添加第二个条件）
  - 条件2：`适配度（AI）` = `中`
  - **逻辑关系**：满足任一条件即可（或）

### 2. AI提示词

```
你是文字梗挖掘专家。从标题中识别文字游戏潜力。

【识别类型】
1. 成语改编（如"不负众望" → 真的不负/辜负众人期望）
2. 谐音梗（如"走漏风声" → 口袋漏风了）
3. 字面理解（如"胖的人" → 被打成平）

【正面案例】
- "铁杵磨成针" → 用铁杵打人（字面理解）
- "无法企及的帅" → 帅到无法企鹅/站立（谐音）
- "省钱的秘诀" → 钱都没了，还省啥钱（反转）

【输入】
标题：{标题}

【输出】
如果有文字梗潜力，输出改编思路（20字以内）
如果没有，输出：无
```

### 3. 其他设置
- **是否进行深度思考**：Medium（中度思考）
- **图片内容**：不勾选
- **视频内容**：不勾选
- **自定义行数**：5
- **自动更新**：不勾选

---

## 测试验证

配置完成后，找几条不同适配度的热搜测试：

| 标题 | 适配度（AI） | 预期结果 |
|------|-------------|---------|
| 不负众望的表现 | 高 | 成语改编：真的不负众人期望 |
| 明星恋情曝光 | 中 | 无（或有文字梗则生成） |
| 国际政治局势 | 低 | （不生成，字段为空） |

---

## 注意事项

1. **触发条件很重要**：必须设置为"适配度=高 或 中"，否则会对所有记录生成，浪费算力
2. **复制完整Prompt**：确保没有遗漏字段引用 `{标题}`
3. **3个表都要配置**：微博、抖音、今日头条都需要添加此字段

---

## 配置后效果

```
数据采集 → 写入热搜表
   ↓
适配度（AI）自动生成
   ├─ 高 → 触发文字梗挖掘 + 老李金句
   ├─ 中 → 触发文字梗挖掘 + 老李金句
   └─ 低 → 跳过（不生成任何内容）
```

这样可以节省算力，只为有价值的话题生成文字梗分析。
