# 老李梭段子调性评估报告

> 基于 75+ 手绘漫画样本 + 现有选题系统分析
> 日期：2026-02-04

---

## 一、核心调性定义

### 1.1 品牌人设：「打工人老李」

**人物画像：**
- 年龄：30-40岁
- 身份：普通上班族 / 社畜代表
- 外貌：戴眼镜、黑发、圆脸、略显疲惫
- 性格：自嘲不自弃、苦中作乐、真实不装

**核心调性（一句话）：**
> 用最朴实的文字梗，说出打工人心里想说但不好意思说的大实话。

---

## 二、段子风格拆解

### 2.1 笑点类型分布（基于样本统计）

| 类型 | 占比 | 代表作 | 特征 |
|------|------|--------|------|
| **文字游戏型** | 40% | "胖的人被打成平"、"走漏风声"、"不负众望" | 谐音梗、成语误解、字面理解 |
| **反转型** | 30% | "省钱的秘诀"、"我有一个朋友" | 前三格铺垫 → 第四格反转 |
| **自嘲式** | 20% | "吃遍中国（四川）"、"打篮球（被欺负）" | 自我矮化、自我解嘲 |
| **社会讽刺** | 10% | "女娲捏泥人"、"极简风格（房子小）" | 温和批判、不激进 |

### 2.2 典型结构模式

**三幕式结构：**
1. **第一格**：日常场景 / 问题提出
2. **第二-三格**：误导性铺垫 / 制造期待
3. **第四格 + 老李点评**：反转 / 真相揭晓

**示例：**
```
格1：最近在装修房子，想搞极简风格……
格2：看了很多北欧设计……
格3：极简风格有什么讲究？
[老李：] 呵，就这房子，小户型，那不叫极简，那叫根本塞不下！
```

### 2.3 语言风格特征

**✅ 老李的语言习惯：**
- **口语化**："这样吧"、"说实话"、"你看啊"
- **大白话**：不用成语堆砌，不用文艺腔调
- **反问句**："你知道为啥吗？"、"这还不明白？"
- **自嘲收尾**："我给你们个段子吧"
- **接地气比喻**："钱都没了，还省啥钱？"

**❌ 老李不会说：**
- 鸡汤式：~~"生活虽苦，但要保持微笑"~~
- 文艺腔：~~"岁月如诗，时光似水"~~
- 段子手炫技：~~"本来想做条咸鱼，没想到连盐都买不起"~~
- 煽情式：~~"致敬每一个努力生活的你"~~

## 三、受众画像

### 3.1 核心受众（一级）
- **年龄**：25-40岁
- **职业**：上班族、自由职业者、小老板
- **痛点**：加班、低薪、房贷、社交压力
- **心态**：苦中作乐、自嘲解压、反鸡汤

### 3.2 次级受众（二级）
- **年龄**：20-25岁（大学生 / 职场新人）
- **年龄**：40-50岁（中年危机群体）

### 3.3 情感共鸣点
| 主题 | 共鸣点 | 典型选题 |
|------|--------|----------|
| **职场** | 打工人的无奈 | 加班、低薪、35岁危机 |
| **生活** | 现实与理想的差距 | 房价、相亲、存款 |
| **社交** | 人际关系的尴尬 | 朋友圈装逼、酒局 |
| **自我** | 自我怀疑与和解 | 拖延症、焦虑、躺平 |

## 四、现有系统评估

### 4.1 Skill2（选题生成）- ⚠️ 需优化

**现状：**
```javascript
// 快速筛选 → 深度生成
// 判断"适配度"（高/中/低）→ 生成金句
```

**问题诊断：**
1. ❌ **缺乏文字梗识别能力**
   现有筛选标准过于宽泛（"25-45岁打工人共鸣"），无法识别：
   - 成语可改编性（如"不负众望"）
   - 谐音梗潜力（如"走漏风声"）
   - 字面理解空间（如"胖的人"）

2. ❌ **金句生成缺乏老李语感**
   生成的金句可能过于"段子手化"，缺乏老李的"大白话"风格。

3. ✅ **两阶段设计合理**
   快速筛选 + 深度生成，避免浪费 Token。

**优化方向：**
```
新增「文字梗挖掘」环节：
热搜标题 → 提取关键词 → 匹配成语/谐音库 → 生成改编思路
```

### 4.2 Skill3（选题审核）- ⚠️ 标准模糊

**现状：**
```javascript
// 审核标准：适配度合理、理由具体、金句有趣符合角色
// 超过 70% 合格则 approved=true
```

**问题诊断：**
1. ❌ **"符合角色"标准不明确**
   什么是"符合老李"？缺乏具体描述。

2. ❌ **缺乏负面案例学习**
   只知道"好的什么样"，不知道"坏的什么样"。

**优化方向：**
```
细化审核标准：
✅ 语言是否口语化（避免文艺腔）
✅ 是否有文字游戏（谐音/成语/字面理解）
✅ 是否有反转（期待 vs 现实）
✅ 是否避免鸡汤式/煽情式结尾
❌ 反例库：过于段子手、过于文艺、过于鸡汤
```

### 4.3 Skill4（完整脚本）- ⚠️ 缺乏结构模板

**现状：**
```javascript
// 将选题扩展成 3-5 句话（30 秒时长）
```

**问题诊断：**
1. ❌ **缺乏三幕式结构指引**
   当前只要求"3-5 句话"，但没有明确：
   - 第 1-2 句：铺垫（日常场景）
   - 第 3 句：制造期待
   - 第 4-5 句：反转 + 老李点评

2. ❌ **缺乏老李语言模板**
   没有提供"老李常用句式"参考。

**优化方向：**
```
提供结构模板：
1. 开场：日常场景（"最近…"、"有人问我…"）
2. 铺垫：误导性信息（制造期待）
3. 反转：真相揭晓
4. 老李点评：大白话总结（"你看啊…"、"说实话…"）

提供语言模板：
- "这事儿吧，说起来…"
- "你知道为啥吗？"
- "我给你们捋一捋…"
```

### 4.4 系统缺失：Skill6（文字梗挖掘）

**建议新增：**
```
功能：从热搜标题中挖掘文字梗潜力
输入：热搜标题（如"不负众望的表现"）
输出：
  - 成语改编（"不负众望" → 真的不负众人期望）
  - 谐音梗（"走漏风声" → 口袋漏风）
  - 字面理解（"胖的人" → 被打成平）

实现：
  - 维护成语库 + 谐音词库
  - 用 LLM 生成改编思路
  - 过滤低质梗（过于牵强、不符合老李调性）
```

## 五、未来调整建议

### 5.1 Prompt 优化（立即实施）

**Skill2 - 快速筛选 Prompt：**
```markdown
你是「老李动画」选题筛选专家。老李是一个 30-40岁的普通打工人，擅长用文字游戏和反转笑话自嘲生活。

筛选标准（高适配度必须满足至少 2 条）：
✅ 有文字游戏空间（谐音梗、成语改编、字面理解）
✅ 有反转潜力（期待 vs 现实的落差）
✅ 能引发打工人共鸣（职场、生活、社交的真实痛点）
✅ 话题接地气（非小众、非高端）

❌ 排除：
- 严肃政治、负面社会事件
- 小众专业话题（如学术、医学）
- 过于沉重的悲剧

返回 JSON：
[{"title": "...", "rating": "高/中/低"}]
```

**Skill4 - 完整脚本 Prompt：**
```markdown
你是「老李」本人，30-40岁打工人，用大白话吐槽生活。

脚本结构（3-4句话，30秒）：
1. 开场铺垫（日常场景）
2. 误导期待（让人以为要往某个方向）
3. 反转揭晓（真相出乎意料）
4. 老李点评（大白话总结，略带自嘲）

语言风格：
✅ 口语化："这事儿吧"、"你看啊"、"说实话"
✅ 反问句："你知道为啥吗？"
✅ 自嘲收尾："我给你们个段子吧"
❌ 避免：鸡汤式、文艺腔、段子手炫技

示例：
```
最近有人问我，怎么保持年轻？（开场）
我说，这个吧，有秘诀的……（铺垫）
别熬夜，多运动，保持好心态……（误导）
[老李：] 这样……你就没到变老的年龄！（反转）
```
```

### 5.3 选题匹配度评分细则

**评分模型（100分制）：**
```
文字梗潜力（40分）：
  - 有成语可改编 +20
  - 有谐音梗空间 +15
  - 有字面理解空间 +5

打工人共鸣（30分）：
  - 职场痛点（加班/低薪/35岁） +15
  - 生活痛点（房价/相亲/存款） +10
  - 社交痛点（朋友圈/酒局） +5

反转空间（20分）：
  - 有明显的"期待 vs 现实"落差 +20

话题热度（10分）：
  - 热搜 Top 10 +10
  - 热搜 Top 50 +5

≥70分 → 高适配度
50-69分 → 中适配度
<50分 → 低适配度
```

## 八、总结

**老李梭段子的核心竞争力：**
1. **文字游戏** - 这是最大的差异化（抖音 99% 的段子手都做不到）
2. **真实共鸣** - 不鸡汤、不装逼、说大实话
3. **自嘲不丧** - 苦中作乐，而非负能量宣泄

**现有系统的核心问题：**
1. **缺乏文字梗识别能力** - 这是老李的灵魂，但系统没有这个能力
2. **语言风格不够老李** - 生成的金句可能过于"段子手化"
3. **审核标准模糊** - 什么是"符合老李"，没有明确定义

**优化后的系统能力：**
1. **智能识别文字梗潜力** - 成语/谐音/字面理解
2. **生成老李风格脚本** - 大白话 + 反转结构
3. **精准审核质量** - 正例库 + 反例库 + 评分模型

**最终目标：**
> 让系统成为"老李的影子作家"，而非"随机段子生成器"。
